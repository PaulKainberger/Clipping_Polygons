File

Here we describe the file format used by this program to save and open polygons.

Specification

The program provides a possibility to save polygons in files. Such a file will have the following structure:

  1. Each line stands for a point in a polygon.
  2. The x and y-coordinate are separated by commas.
  3. Distinct polygons are separated by blank lines.

Example

Example for a file with a triangle and a quadrangle:

1.5,2.12
-7.4,5.3
5,-1

4,2.3
0,-5.76248359
12,-5
1,1
